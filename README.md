# Pure Water Tracker (General Edition)

**「メーカー公表値を超えて、純水器の『真の実力』を可視化する。」**

本アプリは、純水器のメーカーやモデルを問わず、ユーザー自身が数値をカスタム入力することで、純水生成量やコストパフォーマンスを正確に把握するための管理ツールです。

## 🌟 開発コンセプト：純水器ビジネスの不透明さに光を
現在、市販されている純水器の「生成量」や「寿命」の基準は、メーカーごとにバラバラであり、必ずしもユーザーの使用環境に即しているとは言えません。

- **統一基準のない公表データへの疑問**: 自社に都合の良い条件下でのデータ公表を是正したい。
- **樹脂の質と価格の相関性の可視化**: 樹脂効率など、質に伴う適正価格を明らかにしたい。
- **ユーザー主導による「真の樹脂効率」の蓄積**: 純水器ビジネスの「闇」を可視化し、皆さんに還元する。

本アプリで得られた実測データを集約・還元していくことで、ユーザーが根拠を持って純水器や樹脂を適正に選べる世界を目指しています。

## ✨ 特徴
- **自由なカスタム設定**: 樹脂量やTDS値を入力することで、あらゆるメーカー品・自作システムに対応。
- **データ駆動型の管理**: 独自の計算ロジックにより、現在の樹脂効率をリアルタイムで算出。
- **汎用性への注力**: 特定の製品に依存せず、業界全体のデータを比較・分析するためのベースとなります。
